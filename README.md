### About me
- 🎙️ I build **virtual employees** — AI assistants that join **Microsoft Teams** voice &amp; video calls as real participants, so people get their meeting hours back.
- 🤖 Beyond Teams, I build **AI assistants &amp; agents** broadly — **RAG**, **MCP** tool-use, multi-agent orchestration, and **Microsoft 365** productivity integrations (Graph / Planner / To Do).
- 🧠 Deep in **real-time agentic AI**: speech-to-speech, streaming STT → agent → TTS, live vision over camera &amp; screen-share, and barge-in.
- ☁️ I take it to production: the **Azure** media plane (Graph Communications / application-hosted media) and the **multi-tenant SaaS** architecture underneath.
- 🏛️ **Senior Architect – Digital Transformation** by day, hands-on builder by night — I like owning a system from the **architecture &amp; design** down to the code.
- 🌍 Based in **Dubai, UAE** · I speak **English &amp; Arabic**.
- 💬 Ask me about **real-time voice agents, Microsoft Teams calling bots, LLM tool-use, workflow automation, and multi-tenant SaaS on Azure**.

### 🔭 Featured work

**[OpenClaw — Microsoft Teams CVI](https://github.com/ahenawy/openclaw)** — I build the **Teams voice &amp; video integration**: an AI assistant that joins live calls with dual realtime/streaming paths, continuous vision, "speak-only-when-addressed" wake-phrase gating, outbound call-backs with voicemail, and an avatar with expression &amp; lip-sync cues. Security-first — recording-gated, allowlist-closed-by-default, HMAC-signed media.
&nbsp;&nbsp;`TypeScript · Realtime STT/TTS · LLM agents · Microsoft Graph`

**Teams Media SaaS Bridge** *(private)* — a **.NET application-hosted-media worker** bridging Teams calls to an AI backend over an HMAC-authenticated WebSocket, evolving into a multi-tenant, bring-your-own-identity SaaS on **Azure VMSS + Key Vault**.
&nbsp;&nbsp;`C# · .NET 8 · Microsoft.Graph.Communications.Calls.Media · Azure`

**[langgraph-livekit-agents](https://github.com/ahenawy/langgraph-livekit-agents)** · **[Open-Generative-AI](https://github.com/ahenawy/Open-Generative-AI)** — LangGraph + LiveKit realtime voice/video agents, and self-hosted / open generative-AI models (Qwen-Omni · MLX · vLLM).

**[nestjs-cambpm](https://github.com/alaamh/nestjs-cambpm)** *(@alaamh)* — NestJS connector for the **Camunda BPM** workflow engine (contributed NestJS v9 support) — process automation for agentic and enterprise backends.
